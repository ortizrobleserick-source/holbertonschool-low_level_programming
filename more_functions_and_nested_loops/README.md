# More Functions and Nested Loops

## Project Overview

This project focuses on writing various functions that work with loops, including checking if characters are uppercase or lowercase, printing numbers, and performing other basic operations. It aims to improve understanding of basic C concepts such as loops, conditionals, and function declarations.
